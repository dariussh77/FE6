Moduliai

Moduliai (moduliarizacija) - kažkokių funkcijų/kintamųjų/klasių... laikymas atskiruose failuose ir jų export'avimas ir import'avimas.
Moduliai naudojami tam, kad visko nelaikyti viename faile, kas projektui plečiantis gali tapti labai sudėtinga susigaudyti faile.

Export - suteikiama galimybė kažkokioms funkcijoms/kintamiesiems/klasėms... būti pasiekiamiems iš kitų failų.
export default name - eksportuojamas pagrindinis failo elementas. Gali būti tik vienas defaultinis export'as per vieną failą.
export { name1, name2, ... } - eksportuojami papildomi elementai. Galima eksportuoti kiek nori, juos rašant į {} ir atskiriant kableliais.

Import - suteikia galimybę pasiekti kažkokias funkcijas/kitamuosius/klases... iš kitų failų.
import anyNameYouWant from './fileRouteAndFile' - importuojamas pagrindinis nurodyto failo elementas. Jį galime importuoti kokiu tik patys norime vardu.
import { name1, name2, ... } from './fileRouteAndFile' - importuojami TIK nurodyti papildomi elementai. Importuojant papildomus elementus, reikia nurodyti jų specifinius vardus.
import { name1 as somethingElse, name2, ... } from './fileRouteAndFile' - po importuojamo elemento pavadinimo parašius žodelį "as" galima nurodyti kokiu kitu pavadinimu į tą elementą šiame faile kreiptis.
import anyNameYouWant, { name1 as somethingElse, name2, ... } from './fileRouteAndFile' - galima importuoti IR default IR paprastą IR paprastą pervadintą...
