Kintamieji

Kintamieji skirti saugoti/laikyti įvairią informaciją programos veikimo metu.

  • string - tekstas
  • number - skaičius
  • boolean - loginis
  • array - masyvas
  • object - objektas

Aritmetika

Kažkokie veiksmai atliekami tarp duomenų/kintamųjų.

  • + - sudėtis
  • - - atimtis
  • * - daugyba
  • / - dalyba
  • ** - kėlimas laipsniu
  • **(1/y) - šaknies traukimas
  • % - liekanos gavimas

Palyginimai

Palygini keletą duomenų. Ir palyginimas grąžina true arba false.

  • > - daugiau
  • < - mažiau
  • >= - daugiau arba lygu
  • <= - mažiau arba lygu
  • == - lygu (reikšmės)
  • === - lygu (reikšmės ir tipai)
  • != - nelygu (reikšmės)
  • !== - nelygu (reikšmės ir tipai)
  • ! - (NOT) - apverčia boolean reiškmę
  • && - (AND) - tikrina ar abejose pusėse sąlyga grąžina true
  • || - (OR) - tikrina ar bent vienoje pusėje sąlyga grąžina true

Sąlygos

Kodas sąlygų viduje vykdomas tik tokiu atveju, jeigu sąlyga grąžina tiesą.

  • if(){} else if(){} else{}
  • switch() case:
  • ? : - (ternary)
  • && - (logical AND)

Funkcijos

Funkcijose esamas kodas vykdosi TIK tuomet, kai funkcija yra iškviečiama.
Funkcija gali būti iškviečiama daugelį kartų.
Funkcija priima parametrus (duomenis), su kuriais atlieka kažkokius veiksmus. Kreipiantis į funkciją yra perduodami argumentai (duomenys).
Funkcija gali grąžinti kažkokį rezultatą.
Funkcijos gali būti perduotos kaip parametrai ir tuomet iškviečiamos; tai vadinama - callback.
callback'ai dažniausiai aprašomi kaip bevardės funkcijos. Tokiu atveju jų nereikia perduoti kaip parametrų.

  • function name(){} - paprasta funkcija
  • const name = () => {} - arrow/lambda funkcija
  • function(){} - bevardė funkcija
  • () => {} - bevardė arrow funkcija

NODE

Selektinimas

Iš JS kreipiamasi į HTML ir selektinama (taip pat kaip ir CSS'e) kažkoks arba kažkokie elementai.

  • querySelector('selektorius') - grąžina pirmą rastą elementą kaip Node'ą
  • querySelectorAll('selektorius') - grąžina visus rastus elementus kaip NodeList'ą(beveik tas pats kas masyvas)
  • Senieji:
    • getElementById('id')
    • getElementsByClassName('class')
    • getElementsByTagName('tagName')

Event'ai

Laukiama kol kažkokiam NODE elementui įvyks nurodytas veiksmas (pvz.: paspaudimas, pelėsUžėjimas...) ir tuomet vykdoma nurodyta funkcija.

NODE.addEventListener('click', funkcija)

Kūrimas

Galima kurti ir įkelti HTML elementus per JS'ą.
Tiems sukurtiems elementams, galima suteikti įvairias savybes (event'us, atributus, kitusElementus...)

Komandos

  • NODE.createElement()
  • NODE.createTextNode()
  • NODE.append()
  • NODE.appendChild()
  • NODE.classList.add()
  • NODE.classList.remove()
  • NODE.classList.contains()
  • NODE.setAttribute()
  • NODE.remove()
  • ...

Formų nuskaitymas

Forma yra nuskaitoma po to kai įvyksta formos submit event'as.
Per JS'ą, sulaukę formos pateikimo, sustabdome puslapio persikrovimą su event.preventDefault() ir susirenkame formos įvesčių laukuose pateiktą informaciją.
Pateiktą informaciją susirenkame kreipdamiesi į formos elementų vardus/id.

Metodai

Metodas yra funkcija, priskirta kažkuriam specifiniui objektui/masyvui.

Masyvų metodai

Funkcijos skirtos tik masyvams. Jų yra labai daug ir įvairių.
Masyvai turi iteracinius metodus. Jie skirtu sukti ciklą per visą masyvą ir atlikti kažkokius veiksmus su kiekvienu masyvo elementu.

Syntax

masyvas.iteracinisMetodas((masyvoElementas, iteraciją, visasMasyvas) => { veiksmai... })

Metodų įvairovė

  • String
  • Number
  • Math
  • Array
  • Object
  • Date
  • JSON
  • Global_Objects...

Destruktūrizavimas

Išardyti objektą/masyvą/gaunamusParametrus... į mažesnes dalis.

Į funkciją/metodą/klasę gaunamas vienas DIDELIS parametras ir iš jo galima pasiimti tik reikalingas reikšmes.
Galima spread'inti masyvą į atskirus gabaliukus.

Klasės

Klasės yra skirtos kurti objektams.
Klasei galima perduoti argumentus, kuriuos klasė konstruktoriaus viduje priima ir pasivadina kažkokiais pavadinimais, kuriuos naudoja visos klasės viduje.

Pagrindiniai klasės principai

  • Inkapsuliacija - viskas ko klasei reikia yra jos viduje
  • Abstrakcija - supaprastina kuriamo objekto naudojimą
  • Paveldimumas - kuriant klases galima paveldėti kitas klases
  • Polimorfizmas - tos pačios klasės objektai, gali turėti skirtingas struktūras

Fetch

Fetch skirtas tam, kad būtų galima kreiptis į kokį nors API ir iš jo perduoti/pasiimti/redaguoti/trinti duomenis.

Metodai

  • GET - gauna duomenis
  • POST - nusiunčia duomenis
  • DELETE - trina duomenį pagal ID
  • PATCH - lopyja duomenį pagal ID
  • PUT - perrašo duomenį pagal ID
