JASON serveris

  1. Susiinstaliuojame iš: https://nodejs.org/en - pasirenkame LTS download

  2. Sukuriame JSON serveri- Paleidziame konsoleje: npm install -g json-server

  3. Patikrinti comand eilutėje: json-server -v

  4. Susikuriam data.json faila

  5. Iskonsolinti :Json.stringify(data) - ir paimti is Devtoool

  6. Ikeliame is Devtool i Jasonlin.com ir format

  7. Is Jsonlint.com Nukopiname i data.json

  8. Naviguojame iki data.json failiuko - komandinėje eilutėje

  9. Paleidziame data.json pajungima - :$ json-server --watch data.json

  10. Atsidarome narsykleje Json serverio puslapi : (is command) http://localhost:3000

JSON metodai

Get metodas

fetch('http://localhost:3000/automobiliai')
.then(res => res.json())
.then(data => {
console.log(data);
})

Get specific metodas

fetch('http://localhost:3000/automobiliai/2') // 2- antras ID Nr
.then(res => res.json())
.then(data => {
console.log(data);
})

DELETE metodas - galima pasidaryti begalini ciklą, jei kreipiamės tesiogiai, o ne per listening

fetch('http://localhost:3000/automobiliai/2', {method: 'DELETE'});

POST metodas - galima pasidaryti begalini ciklą, jei kreipiamės tesiogiai, o ne per listening

fetch ( 'http://localhost:3000/automobiliai' ,
{method:'POST',
header: {'Content-Type' : 'application/json'} ,
body: JSON.stringify({marke: 'BMW' , modelis: '6' ,gamybosMetai : 2020 })
})
