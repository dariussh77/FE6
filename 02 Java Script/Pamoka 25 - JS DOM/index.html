Document Object Model

NODE

NODE'as - HTML elementas (JS'o aplinkoje).

Daug geresnis būdas kurti HTML elementus per JS yra ne užpildant tėvinio elemento innerHTML'ą kažkokia informacija, bet kuriant informacijos vienetus kaip kintamuosius ir priskiriant juos vienus kitiems.

NODE Metodai

  • createElement('tagVardas')
  • kam.append(ką, kąDar, kąąąDaaar...)
  • kam.appendChild(ką)
  • createTextNode('tekstas...')
  • setAttribute('name', 'value')
  • classList
    • .add('naujaKlase')
    • .remove('esamaKlase')
    • .replace('esamaKlase','naujaKlase')
    • .toggle('kazkokiaKlase') - jeigu nėra - prideda, jeigu yra - išima
