Duomenu Saugojimas Vartotojo Dalyje

Cookies,Session, Local - informacija yra saugoma jūsų įrenginyje/aplikacijoje. Skirtingų svetainių informacija suagoma atskirai

Cookies

Svetainėje saugomi duomenys.
Saukoma iki 4kb
Reikia nustatyti iki kadasaugoti
Siunčiami kartu su requestais į backendą.
Sintaksės Pvz: document.cookie=`Pastatas=Namas; expires=${new Date('2023 04 01 14:00:00')}`

Session storage- pasibaigus sesijai ištrina

window.sessionStorage.setItem('1 savaitės diena', 'pirmadienis');
window.sessionStorage.setItem('2 savaitės diena', 'antradienis');
window.sessionStorage.removeItem ('1 savaitės diena');
window.sessionStorage.clear() //išvalo visas reikšmes;

Local storage - laiko visa laika duomenis

window.localStorage.setItem('1 savaitės diena', 'pirmadienis');
window.localStorage.setItem('2 savaitės diena', 'antradienis');
window.localStorage.removeItem ('1 savaitės diena');
window.localStorage.clear() //išvalo visas reikšmes;br

JSON - Java Script Object Notation

JavaScipt duomenų failas. objekto keyjus imeta i kabutes.

JSON.stringify -pavertimas į JSON formatą- window.localStorage.setItem('duomenys', JSON.stringify(duomenys));
JSON.parse - duomenų ištraukimas iš JSON - console.log(JSON.parse(window.localStorage.getItem('duomenys')))
