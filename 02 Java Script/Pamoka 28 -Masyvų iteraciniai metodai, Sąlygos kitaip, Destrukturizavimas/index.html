ForEach metodas

Syntax:

forEach((element)=>{veiksmai})a forEach((element,i)=>{veiksmai}) forEach((element,i,array)=>{veiksmai})

Suka ciklą per visą masyvą ir kiekvienos iteracijos metu atlieką veiksmus su tos iteracijos masyvo elementu. ForEach neturi return

Filter metodas

Syntax:

filter((element)=>{veiksmai}) filter((element,i)=>{veiksmai}) filter((element,i,array)=>{veiksmai})

Suka ciklą per visą masyvą ir tikrina ar i-tas atitinka sąlygas, jei tenkina, jis gražinamas, jei ne negrazinamas- yra return

MAp metodas

Syntax:

map((element)=>{veiksmai}) map((element,i)=>{veiksmai}) map((element,i,array)=>{veiksmai})

Suka ciklą per visą masyvą ir kiekvienos iteracijos metu kažkaip koreguoja masyvo elementą ir gražina į naują masyvą

Reduce metodas

Syntax:

reduce((acumulator,currentValue)=>{veiksmai}) reduce((acumulator,currentValue,i)=>{veiksmai}) reduce((acumulator,currentValue,i,array)=>{veiksmai}) reduce((acumulator,currentValue)=>{veiksmai},initialValue) reduce((acumulator,currentValue,i)=>{veiksmai},initialValue) reduce((acumulator,currentValue,i,array)=>{veiksmai},initialValue)

Suka ciklą per visą masyvą ir iteracijos metu su accumulator(reikšmė, kurią gražino praeitas ciklas) atlieka kažkokį veiksmą, naudodamas šios iteracijos elementą(masyvas[i]- currentValue). Metodo gale, gražina vieną reikšmė(visų masyvo elemntų rezultatą). jeigu yra nurodyta pradinė reikšmė ir pirmuoju masyvo elementu. Jeigu nėra nurodyta InitialValue tuomet pirmosios itearacijos metu yra atliekami su pradine reikšme ir antrąją masyvo reiksme

Įmantrios sąlygos

Ternary(?:)

Ternary - sąlygos sutrumpinimas su ?: zr.pavyzdi:

kintamasis1>x?veiksmas_true:kintamais2>y?veiksmas_else_if_true:veiksmas_else;

Logical and &&

Kaip ternary vietoj ? - &&, o else ir else if nebėra:

kintamasis1>x && veiksmas_true(kitų veiksmų negali būti)

Destruktūrizavimas

Elementų reikšmių išskirtymas dalimis. Arba visų masyvo duomenų pavertimas kintamaisiais

Masyvų greitas išskaidymas pavieniais kintamaisiais:

let [d0,d1,d2='default, jei nėra duomens',...likucioMasyvas]=masyvas

Objektų greitasis iškirtymas pavieniai kintamaisiais:

let {key0:suteikiamasVardas, key1:{key1_1:suteikiamasVardas2},key3='DEfault, jei nėra duomens', ...likutisObjekto}=objektas;

Kintamųjų duomenų sukeitimas vietomis

[a,b]=[b,a];
