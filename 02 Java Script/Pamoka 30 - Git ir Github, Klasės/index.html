Git komandos: git (tarpelis)

  • init -sukuria git'repozitorija
  • add nurodytiFailai- pridėti koreguotus/kurtus/naujus failus/aplankalus prie paruoštų failų komitui
  • commit -m - iš paruoštų(staged) failų ir aplankalų sukuriama kodo versija su atitinkama žinute(message(-m)) nusakančia kuo tas kodas pakito
  • status- grazina informacija apie sattusa
  • push - nusiunčia comitus i išorinės repozitoriją(Github)
  • pull - ištraukia comitus iš išorinės repozitorijos(Github)

Git konfigūracija:

  • git config --global user.name "Tavo varas iš Github- neprivalyu
  • git config --global user.email "Tavo mailas iš Github - priivalu
  • git config --global core.editor "code" -VSCode
  • git config --global core.autocrlf true (vietoje true -input -MAc ar Windows atveju) -
  • git config --global -e -atidarys editoriu
  • git config --help -atidarys pagalba
