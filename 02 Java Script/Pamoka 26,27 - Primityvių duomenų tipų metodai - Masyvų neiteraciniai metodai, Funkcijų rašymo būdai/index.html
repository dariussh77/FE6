Metodai

Metodai - jaus sukurtos funkcijos, kurios priklauso kažkokiam specifiniam elementui.

Metodai skirti tam, kad prageitintų/palenvintų programuotojų darbą.

Boolean Metodai

  • Boolean.toString() - paverčia į tekstą 'true'arba 'false'
  • Boolean.valueOf() - tiesiog išveda true arba false naudojant truthy ar falsy išraiškose

Number Metodai

Propieties:

  • Number.MAX_SAFE_INTEGER - išveda maksimalų saugus sveikasis skaičių 2laipniu53-1
  • Number.MIN_SAFE_INTERGET - išveda maksimalų saugus sveikasis skaičių 2laipniu53-1
  • Number.MAX_VALUE - maksimalus skaičius, kuri padidinus *2 gauname Infinity
  • Number.POSITIVE_INFINITY - teigiama begalybė
  • Number.NEGATIVE_INFINITY - BEGALYBĖ į minus
  • Number.MIN_VALUE-minimalus skaičius, kuri padidinus /2 gauname Infinity
  • Number.EPSILON - išveda ypač mažą skaičių
  • Number.NaN - išvedamas NaN - ne skaičius

Metods:

  • Number.isFinite() - ar baigtinis skaičius duoda atsakymą true arba false (2/3- baigtinis)
  • Number.parseInt- konvertuoja į skaičių išveda tik sveikąją dalį, neapvalina.
  • Number.parseFloat - konvertuoja į skaičių.
  • .toFixed(n) - suapvalina iki n-to skaičiaus po kalblelio 5.6666.tofixed(2)=5.67
  • Number.isInteger(3.6) - ar sveikasis true/false
  • ...
