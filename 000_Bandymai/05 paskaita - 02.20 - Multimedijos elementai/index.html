Klaviatūros komandos VS code arba PC

  1. CTRL C - kopijuoja pažymėtą informaciją
  2. CTRL V - įklijuoja kopijuotą informaciją
  3. CTRL X - iškerpa pažymėtą informaciją
  4. CTRL ENTER - naujos eilutės sukūrimas apačioj
  5. CTRL SHIFT ENTER - naujos eilutės sukūrimas viršuj
  6. CTRL S - saugoti failą
  7. CTRL Z - atšaukti paskutinį veiksmą/us
  8. CTRL SHIFT Z / CTRL Y - atšaukia CTRL Z komandą/as
  9. CTRL D - parenka sekantį vienodą pažymėtą elementą
  10. CTRL F2 - parenka visus vienodus pažymėtus elementus
  11. CTRL A - pažymėti viską
  12. CTRL F - atsidaro paieškos eilutę
  13. CTRL SHIFT F - atidaro paieškos įrankį (per visus failus)
  14. ALT MouseClick - Pažymi vietas, kur kuriami papildomi kursoriai
  15. ALT arrowUp/arrowDown - perkelia pažymėtą/as eilutę/eilutes aukštyn/žemyn
  16. ALT/Option SHIFT arrowUp/arrowDown - kopijuoja pažymėtą/as eilutę/eilutes aukštyn/žemyn
  17. CTRL / - Sukuria komentarą pažymėtose eilutėse
  18. ALT SHIFT F - Automatinis formatavimas
  19. CTRL -> / CTRL <- - kursorius peršoka per visą žodį
  20. FN -> / FN <- - kursorius peršoka per visą eilutę
  21. ALT TAB - vaikščioti per atidarytus langus kompiuteryje
  22. WINDOW TAB - vaikščioti tarp virtualių ekranų kompiuteryje
  23. ALT F4 - išjungia atidarytą programą
  24. CTRL F4/W - išjungia atidarytą langą/tab'ą (programos)
  25. CTRL SHIFT ESC - atidaro task manager langą
  26. TAB - tabuliuoti kodą į dešinę pusę
  27. SHIFT TAB - tabuliuoti kodą į kairę pusę
  28. WINDOW L - užrakina vartotojo paskyrą (window)
  29. WINDOW D - atidaro desktop / viską nuleidžia
  30. CTRL T - sukuria naują tab'ą
  31. CTRL TAB/SHIFT TAB - vaikšto per atidarytus tab'us
  32. CTRL SHIFT T - atidaro paskutinį uždarytą langą
  33. WINDOW SHIFT S - dalies ekrano iškirpimas ir saugojimas
  34. CTRL mouseScroll - artina / tolina naršyklės langą
  35. HOME / END - nukelia į eilutės pradžią/galą
  36. CTRL HOME / CTRL END - nukelia į failo pradžią/galą
  37. WINDOW E - atidaro failų naršyklę (File Explorer)
  38. WINDOW R - atidaro run'ą (procesų paleidimo langą)
  39. (dešinysis)ALT - leidžia rašyti kita kalba (kol nuspaustas)
