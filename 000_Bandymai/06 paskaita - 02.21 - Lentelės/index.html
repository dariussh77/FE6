Lentelės tag'ai

  • table - lentelės tag'as
  • thead - lentelės viršus (table head) skirtas stulpelių pavadinimams
  • tbody - lentelės kūnas (table body) skirtas duomenims
  • tfoot - lentelės apačia (table foot) lentelės apibendrinimas
  • tr - lentelės eilutė (table row)
  • td - lentelės duomuo (table data)
  • th - lentelės stulpelio antraštė (table heading)
  • caption - lentelės pavadinimas
  • colgroup - stulpelių grupė
  • col - stulpelis

CSS Selectors

  • element - elemento - pvz.: p
  • class - klasės - pvz.: .className
  • id - id - pvz.: #ID_Name
  • everything - visko - pvz.: *
  • multiple - daugelio elementų - pvz.: div, .pavadinimas
  • inside - elementai elemento viduje - pvz.: div p
  • children - vaikiniai elemento elementai - pvz.: div > p
  • element with class - elementas su klase/id arba klasėmis/id - pvz.: div.hihi.haha#idVardas
  • next element - elementas einantis už elemento - pvz.: input + label
  • next elements - elementai einantys už elemento - pvz.: input ~ label
