GitHub- svetainė kur keliamos kodas Github

  1. Įsirašyti Gitą į PC
  2. Patikrinti Git įrašytas: terminale - git --version
  3. susikuriti Github paskyrą
  4. Konfigūruoti git'ą:
    • git config --global user.name githubVartojoVardas
    • git config --global user.email githubVartojoEmail
    • git config --global core.editor code
    • git config --global core.autocrlf true (Windows) arba input (Linux, mac)
  5. patikrinti ar teisingas config failas - git config --global --edit
