Grid properties

Parent

  • grid-template-columns - nurodome stulpelių kiekį ir plotį - 100px 50px 100px
  • grid-template-rows - nurodome eilučių kiekį ir aukštį - 50px 150px
  • grid-template: grid-template-rows /grid-template-columns;
  • grid-template-area - užvadinami tinklelio langeliai sugalvotais pavadinimais
  • column-gap - tarpas tarp grid'o stulpelių
  • row-gap - tarpas tarp grid'o eilučių
  • gap: row-gap column-gap;
  • justify-items
  • align-items
  • place-items: align-items justify-items
  • justify-content
  • align-content
  • place-content: align-content justify-content
  • grid-auto-columns - automatiškai kuria papildomus stulpelius pagal seką pertekliniam elementų kiekiui
  • grid-auto-rows - automatiškai kuria papildomas eilutes pagal seką pertekliniam elementų kiekiui
  • grid-auto-flow: row | column | row dense | column dense;
  • grid: grid-template-rows /grid-template-columns / grid-template-areas | grid-auto-columns / grid-auto-rows / grid-auto-flow;

Children

  • grid-column-start - nurodo kurioje stulpelio linijoje prasidės elementas
  • grid-column-end - nurodo kurioje stulpelio linijoje pasibaigs elementas
  • grid-row-start - nurodo kurioje eilutės linijoje prasidės elementas
  • grid-row-end - nurodo kurioje eilutės linijoje pasibaigs elementas
  • grid-column: grid-column-start / grid-column-end;
  • grid-row: grid-row-start / grid-row-end;
  • grid-area: name | grid-row-start / grid-column-start / grid-row-end / grid-column-end;
  • align-self
  • justify-self
  • place-self

Special Units & Functions

  • 1fr - nurodo kokią dalį turinio užims (atėmus: tarpus, px, margin, padding, %, ...)
  • repeat(kiek, ką);
  • minmax(nuo, iki);
