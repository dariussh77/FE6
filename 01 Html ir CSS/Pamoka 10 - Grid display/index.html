GridCheatSheet
Grid Garden

Parent

  • grid-template-colums: nurodome stulpelių ir kiekį ir dydį - pvz 100px 50px 100px
  • grid-template-rows: nurodome eilučių kiekį ir aukštį - 50px 150px - paskutiniai duomenų savybės neatsikartoja- pasirenka dydi pagal turinio dydį
  • grid-template: grid-template-rows / grid-template-colums - apjungia auk2iau esan2iu
  • grid-template-area: naudojamas stambiam gridui - užvadinami tikleliio laukai sugalvotais pavadinimais
  • column-gap / row-cap -strulpeliu eilučių tarpai
  • gap: row-cap column-gap pvz :10px 10px -strulpeliu eilučių tarpai
  • justify-items - islygiavimas horizontaliai
  • align-items- išlygiavimas vertikaliai
  • place-items : align-items justify-items
  • justify-content - islygiavimas horizontaliai
  • align-content- išlygiavimas vertikaliai
  • place-content : align-content justify-content
  • grid-auto-row : automatiškai kuria papildomus laukelis pertekliniams elementams. išsorendžia grid-template pabaigos
  • grid-auto-column
  • grid-auto-flow: row column dence
  • grid: grid-template-rows / grid-template-colums | grid-auto-column / grid-auto-row / grid-auto-flow

Children

  • grid-column-start: nurodo kurioje stulpelio linijoje prasidės elementas
  • grid-column-end: nurodo kurioje stulpelio linijoje pasibaigs elementas
  • grid-row-start: nurodo kurioje eilutes linijoje prasidės elementas
  • grid-row-end: nurodo kurioje eilutės linijoje pasibaigs elementas
  • grid-column: grid-column-start / grid-column-end
  • grid-row: grid-row-start / grid-row-end
  • grid-area: name | grid-row-start / grid-column-start /
  • place-self:justify-self align-slef
  • justify-self
  • align-slef

Special Units

  • repeat(kiek kartų, ką)
  • 1fr - sukirto likuti i dalis
  • minmax (nuo, iki)
