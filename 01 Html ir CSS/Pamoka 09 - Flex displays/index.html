Flex Properties

Parent element - container

  • Flex-direction: row / column/row-reverce/ column-reverse
  • fle-wrap: nowrap (default), wrap, wrap-reverce
  • Flex-flow: apjungią flex-direction ir flex-wrap
  • justify-content: (pagrindiniai) flex-start(default), flex-end, center, space-between, space-around, space-evenly
  • align-items (atskirų eilučių nustatymas): strech (default) flex-start, flex-end, center, baseline
  • aligh-content (Viso turinio nustatymui)(no default): strech, flex-start, flex-end, center, space-between, space-around, space evenly (Prioritetas jei maudojamas jei align-items)
  • gap - row-gap column-gap

Children element - flex item

  • Order:number; default (0)
  • align-self (Veikia kaip Aligh-items, bet tik vienam elementui): flex-star, flex-end, auto, center, strech, baseline
  • flex-grow:1 default, padidinti procentaliai
  • flex-shrink:
  • flex-basis: 1px, default auto, nurodo kaip bus paskirstyta nepanaudota vieta
  • flex (apjungimas auksčiau esančių): grow shrink basis
