IKONOS

W3 ikons

Ikonos savo tago neturi, jos naudoja kitus tagus ir priskirdamos jiems klases, būna atvaizduojamos.Dažniausios i (italic) arba span.
Norint naudoti ikonas, reikia prijungti vieną iš dauglio ikonų bibliotekų prie savo sveiatinės. Galimos biliotekos: Font-awesom, bootstrap ( bootstrap ), google, MUI, Ant-Design, Tailwind, Bulma..

Norint stilizuotiikonas jas stilizuojame kaip textinius elemetus: spalva; fonas.

Selektoriai

Selektoriai2 Žaidimas
  • Basic selektoriai
    • Element - elemeto sel: p {
    • class - .classname {
    • id - #id_name {
    • multiple- div, .classname {
    • - parenka elementus elemento viduje bet kuriam gylyje- div p {
    • - parenka elementus elemento viduje tik vienu lygiu zemyn - div > p {
    • elemntas su klase arba klasemis - div.class1.class2#id1 {
    • viskas- bet koks elementas - * {
    • next element - input+label {}
    • elementai esantys uz elementos- input~label {}
  • Atributai:
    • [attribute] -renkasi elementą turintis atributą [classs]
    • [attribute="reikšmė"] -renkasi elementą turintis atributą su reikšme[classs="ggg"]
    • [attribute~="reikšmė"] -renkasi elementą turintis atributą kur viena iš reišmių yra nurodyta[classs~="ggg"]
    • [attribute="reikšmė" -renkasi elementą turintis atributą kur yra nurodyta[classs~="ggg"]
    • [attribute=""] ???????????
    • [attribute^=""] ???????????
    • [attribute^=".https://"] - prasideda reiksme
    • [attribute$=".png"] - pasibaigia
    • [attribute*="fff"] - simboliu seka viduje
