color names

Uždadintų spalvų yra apie 140. rašyti jos pavadinimą.

RGB(a) (Red Grean Blue Alpha)

Naudojant Rgb kodą, kiekviena spalva susideda iš 3pagrindinių spalvų mišinio. Vienos spalvos kiekis nurodymaas kiekvienos spalvo kiekiu nuo 0-255. Melyna rgb(0,0,255)

rgbA - kaip rgb su permatomumu Alpha. Nuo 0 - 1 , žingsnis 0,001/ Pvz Melyna rgb( 0, 0, 255, 0.5)

Hex kodas#

Naudojant hex kodą nurdoma spalva - #00-#FF. Melyna - Pvz #00 00 FF

Permatomomą pridedame šioliktainį skaičių.Pvz #00 00 FF FF pilnai nepermatoma

Hex kodą galima trumpintii, jei yra du vienodi simboliai iš eilės (dvigubai sumažinant)

Gradient
